Use `inquiry-checklist.md` for qualification:

## Phase 6: Close the Sale

### Before Handoff
- [ ] Bill of sale prepared (both parties sign)
- [ ] Title signed and ready to transfer
- [ ] Remove plates (if keeping them)
- [ ] Final payment confirmed (cash, cashier's check, or wire)
- [ ] Do NOT accept personal checks

### At Handoff
- [ ] Walk buyer through operation (tent, awning, heater, fridge, electrical)
- [ ] Hand over documentation packet (receipts, manuals, warranty info)
- [ ] Confirm buyer has brake controller connected
- [ ] Take photo of buyer with trailer (your record)
- [ ] Exchange signed bill of sale

### After Sale
- [ ] Remove listings from all platforms
- [ ] Mark GitHub repo as archived or delete
- [ ] Celebrate
